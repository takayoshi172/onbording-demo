Onboarding トライアル（デモ）

このページは GitHub Pages 上で「オンボーディング」体験を簡単に試すためのテンプレートです。

使い方（概要）

  1. このテンプレートをリポジトリに配置して GitHub Pages を有効化します。
  2. 下の 埋め込みスニペット のコメント位置に、あなたが試したいオンボーディングの SDK/スクリプトを貼り付けます。
  3. コミット＆プッシュ後、公開 URL を開いて挙動を確認します。

埋め込みスニペット

以下のプレースホルダーに、提供されたトライアル用のコード（例：<script> ... </script>、ウィジェット、iframe 等）を挿入してください。

CTA（動作確認用）

オンボーディングの開始トリガーとして利用できるボタンです。
